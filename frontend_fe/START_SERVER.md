# Frontend FE 启动指南

## 启动命令

### 方法 1：使用 Python 服务器（推荐）

```bash
cd frontend_fe
python3 server.py
```

或者从项目根目录：

```bash
python3 frontend_fe/server.py
```

### 方法 2：使用 Python 内置 HTTP 服务器

```bash
cd frontend_fe/public
python3 -m http.server 8000
```

然后访问：`http://127.0.0.1:8000`

**注意**：使用此方法时，需要确保静态文件路径正确。可能需要调整 HTML 中的静态资源路径。

---

## 访问地址

启动服务器后，访问：

```
http://127.0.0.1:8000
```

（端口号可能不同，请查看服务器启动时的输出）

---

## 服务器特性

- **只读服务**：仅提供静态文件服务，无文件写入、无执行、无修改
- **本地访问**：仅绑定到 127.0.0.1，不对外暴露
- **自动端口检测**：自动查找可用端口（8000-8100）

---

## 注意事项

1. 当前实现使用示例数据，实际数据需要从文件系统读取
2. 如需读取实际数据，需要：
   - 创建后端 API（只读）
   - 或直接在浏览器中通过文件系统 API 读取（需要浏览器支持）
   - 或使用现有的 s1_server.py（如果兼容）
