### Run ID Format

`r2_S<scenario>_V<variant>_E<epochs>_S<seed>`

**Example**: `r2_S001_V1_E100_s1337`

### Matrix Dimensions

- **Scenarios**: S-001 to S-012 (12 scenarios)
- **Variants**: V1, V2, V3 (3 variants per scenario minimum)
- **Epochs**: E10, E100, E1000 (3 epoch lengths)
- **Seeds**: s1337, s1338, s1339 (3 seeds)

**Total Combinations**: 12 × 3 × 3 × 3 = 324 runs (minimum)
